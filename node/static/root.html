Welcome to the AI-Powered Medical OCR System API

This is the main information endpoint for the intelligent document processing pipeline. The system extracts and categorizes financial amounts from medical documents using OCR and AI.

How to Use the API

Follow the steps below to authenticate and start processing documents.

Step 1: Create an Account

Endpoint: POST /signup

Description: Create a new user account.

Body (JSON):

{
  "username": "your_username",
  "password": "your_password"
}

Step 2: Log In to Get an Authentication Token

Endpoint: POST /login

Description: Authenticate your credentials to receive a JWT token.

Body (JSON):

{
  "username": "your_username",
  "password": "your_password"
}

Important: The API will return a token in the response. You must include this token in the Authorization header for all protected requests in the next steps.

Header Format: Authorization: Bearer <YOUR_JWT_TOKEN>

Step 3: Process a Medical Document

Endpoint: POST /process_image_stepwise

Authentication: Required (Bearer Token)

Description: Upload a medical document (image) for OCR and AI analysis.

Body (multipart/form-data):

  • Key: file
  • Value: [Select your image file to upload]

Step 4: Retrieve Your Processing History

Endpoint: GET /results

Authentication: Required (Bearer Token)

Description: Fetches all previous processing results associated with your account.
